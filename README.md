# ARPG Demo (Unreal Engine 4)

## Project Overview
This is an ARPG game demo fully implemented using **Unreal Engine 4 Blueprints**.  
It showcases core gameplay mechanics and interactive systems.  
The project demonstrates skills in **Blueprint development, game logic, AI behavior, UI interaction, and IK systems**.

## Project Highlights
1. **Fully Blueprint-based**: Demonstrates proficiency with UE event system, AI, UI, IK systems, and player interactions  
2. **Multi-system integration**: Combines combat, quests, construction, climbing, and online testing  
3. **Extensible**: Can be extended to C++, multiplayer, or more complex game mechanics
