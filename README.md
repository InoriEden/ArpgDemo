# ARPG Demo (Unreal Engine 4)

## Project Overview
This is an ARPG game demo fully implemented using **Unreal Engine 4 Blueprints**.  
It showcases core gameplay mechanics and interactive systems.  
The project demonstrates skills in **Blueprint development, game logic, AI behavior, UI interaction, and IK systems**.

---

## Features

### 0. Game Settings
- Screen settings
- Volume settings
- Custom key bindings

### 1. Map & Navigation
- Map exploration and radar navigation

### 2. Inventory & Construction System
- Player inventory management
- Construction/placement system

### 3. Combat System
- Basic combat mechanics
- Two combat modes (Combat 1 & Combat 2)

### 4. Quest System
- Quest acceptance and completion

### 5. Save & Load
- Game progress save and load

### 6. IK System
- Leg IK: Character alignment with terrain
- Arm IK: Object interaction and animation constraints

### 7. Climbing & Grappling
- Environmental climbing
- Grapple hook interaction

### 8. Steam Online Test
- Simple chat system
- Online functionality test

---

## Tech Stack
- Unreal Engine 4
- Fully implemented using **Blueprints**
- Includes basic AI, event-driven logic, and IK systems

---

## Project Highlights
1. **Fully Blueprint-based**: Demonstrates proficiency with UE event system, AI, UI, IK systems, and player interactions  
2. **Multi-system integration**: Combines combat, quests, construction, climbing, and online testing  
3. **Extensible**: Can be extended to C++, multiplayer, or more complex game mechanics
